# 発信内容設計

プログラミング教室のCanva作業用デザイン仕様メモ

| 項目 | 内容 |
|---|---|
| 配色 | ターコイズ＋ビビッドオレンジの2色。オレンジは今回くすんで見えたので、彩度高め（例：#FF6B35前後）を意識 |
| 3色目 | 使うならターコイズかオレンジの濃淡（トーン違い）のみ |
| NG/OK表現 | 色分けせず、✕／○の記号で判断させる |
| フォント | 見出し・英単語（Scratch, MakeCode等）は視認性の高い欧文フォント、本文は和文ゴシック |
| レイアウト | 表紙1枚→スワイプで詳細→CTA、の流れ。Venn図で「共通点」を見せる構成は今回好評だったので踏襲 |
| ムード | 「楽しそう」＋「ちゃんとしてそう」の両立 |
